# free-iptv-threat-intel

Open-source threat intelligence on malicious domains found in free IPTV playlists.

## Included File

- `scan_results.json` - malicious-only domain verdict database

## Data Policy

- Only domains with `malicious > 0` are published here.
- Clean (`malicious = 0`) and suspicious-only entries are excluded from this public repo.
- Values may change over time as domains are rescanned.

## Schema

Each key is a domain (or host:port) and value follows:

```json
{
  "malicious": 0,
  "suspicious": 0,
  "last_scanned": "2026-03-05T06:36:41+00:00"
}
```

## Notes

- This data is for defensive/security research and risk triage.
- A listed domain is an indicator, not standalone proof of compromise.
- False positives are possible.

## Source

Data is generated by the private `IPTV_playlist_manager` scanning pipeline and synced here automatically.
